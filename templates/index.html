A proposta central do sistema é auxiliar processos de identificação de conteúdos suspeitos, tornando a análise inicial mais organizada, objetiva e tecnologicamente apoiada.

Tecnologia aplicada Processamento de frames, geração de hash e comparação de similaridade.

Sobre o projeto

O sistema foi pensado para atuar como uma ferramenta de apoio à análise de conteúdo, recebendo arquivos GIF, extraindo informações visuais relevantes e comparando esses dados com materiais previamente cadastrados na base do sistema.

🔎

Finalidade

Apoiar a identificação de padrões visuais semelhantes entre arquivos analisados e conteúdos cadastrados como referência, contribuindo para uma triagem inicial mais eficiente.

🧠

Lógica utilizada

A plataforma utiliza processamento de imagem quadro a quadro, transformando frames em hashes perceptuais para medir proximidade visual entre GIFs distintos.

🛡️

Aplicação em segurança

O projeto demonstra como recursos computacionais podem ser usados em contextos de monitoramento, investigação, apoio à triagem e automação de processos em segurança digital.

Como o sistema funciona

O fluxo abaixo resume as etapas executadas pela aplicação desde o recebimento do arquivo até a geração do resultado da análise.

1

Envio do GIF

O usuário seleciona um arquivo GIF para cadastro ou análise dentro da plataforma.

2

Leitura dos frames

O sistema percorre os quadros da animação para identificar suas características visuais.

3

Geração de hash

Cada frame recebe uma representação matemática simplificada para permitir comparação.

4

Resultado comparativo

A aplicação mede a similaridade com a base cadastrada e retorna a melhor correspondência.

Ambiente de operação

Abaixo estão os módulos principais da plataforma: o cadastro de materiais de referência e a análise comparativa de novos arquivos enviados para triagem.

Módulo 01

Cadastrar GIF de referência

Utilize esta área para adicionar um GIF à base comparativa do sistema. Esse conteúdo poderá ser utilizado em análises futuras como parâmetro de semelhança.

Esse cadastro alimenta a base de comparação utilizada pela aplicação.

Módulo 02

Analisar novo GIF

Envie um arquivo para que o sistema realize a leitura dos frames, a extração do hash perceptual e a comparação com os registros existentes na base.

O resultado exibirá a melhor correspondência encontrada e o percentual de similaridade calculado.
