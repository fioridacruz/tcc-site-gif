Como funciona

O sistema recebe um GIF enviado pelo usuário, executa uma análise de similaridade com padrões previamente avaliados e retorna uma classificação visual com apoio à interpretação do resultado.

Objetivo do projeto

Este projeto tem como objetivo apoiar a identificação de GIFs potencialmente maliciosos, fornecendo uma interface acessível, elegante e intuitiva para testes, demonstrações acadêmicas e apresentação profissional em banca.
